05 — Methodology

How BatteryChem-AI Works

An end-to-end data-driven pipeline from experimental data to interpretable AI predictions.

01

Data Collection

Curated experimental datasets from CALiSol-23 database and peer-reviewed literature (JES, JPS, JES_2019, CALiSol). Each record contains solvent system, salt type, concentration, temperature, and measured ionic conductivity.

02

15D Feature Engineering

First-principles topological fingerprints for solvents (MW, TPSA, LogP, HOMO, LUMO), lithium salts (MW, TPSA), and additives (MW, TPSA, LogP, HOMO, LUMO), plus derived delta features (ΔMW, cross-gap).

03

XGBoost Training

Single-output XGBoost regressor with fixed hyperparameters (n_estimators=120, max_depth=6, lr=0.08). Zero formula contamination — labels are pure experimental measurements with Gaussian noise.
